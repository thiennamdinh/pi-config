# Pi Config Repository Instructions

This repository stores Thien-Nam's personal Pi configuration.

Maintenance workflow:
- Push directly to `main`; this repo is only used by Thien-Nam.
- Keep commits concise, well-organized, and logically scoped.
- Do not commit secrets, tokens, session transcripts, local runtime state, or generated caches.
- Prefer documenting durable config/architecture notes in tracked files and excluding machine-local artifacts via `.gitignore`.
